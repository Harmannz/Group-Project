## Getting Started

### Prerequisites
* Node ~5.7.1

### To begin development:   

1. Install node onto system   
2. run: `npm install` to install the node modules
3. run: `npm start` to launch the web app

## Testing

Tests are found in the 'test' folder.
To run the tests, run `npm test`
